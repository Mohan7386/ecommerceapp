# 🛍 Ecommerce App

A modern Flutter Ecommerce application with Firebase backend.

## ✨ Features

- Firebase Authentication
- Google Sign-In
- Cart Management
- Favorites
- Orders
- Responsive UI

## 🛠 Tech Stack

- Flutter
- Firebase
- Provider
